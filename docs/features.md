# Features

## ✔ Automated Job Fetching
Pulls job data from RSS, API, or LinkedIn export.

## ✔ Resume Structuring with AI
Converts raw CV text into clean JSON:
- Skills
- Experience
- Achievements
- Education

## ✔ Intelligent Skill Matching
Matches resume skills with job requirements.

## ✔ Cover Letter Generator
Creates a customized cover letter:
- Based on job description
- Highlights matching skills
- Humanlike writing

## ✔ Google Sheets Integration
Stores all job application data:
- Job metadata
- Match score
- Cover letter
- Application progress

## ✔ Duplicate Job Prevention
No job is processed twice using a hash system.

## ✔ Full Automation Pipeline
Everything runs on auto-pilot with Automation.
